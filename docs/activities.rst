Activities
===========

A Activity is an independent sub-program of the application. (e.g., DataViewer or Settings).
Activities can be developed independently from each other.

Open Add Activity Tab
---------------------
- Click on :ref:`Add Activity Tab <connection>` Symbol
- Choose an Activity
- You may need a connection to a server for this
- Click on Connect
- New activity Tab opens with selected Activity
